### 小小API

- 连接方式（Python）见文件`xxAPItesting.py`即可
```python
import requests
import time

url = "https://v2.xxapi.cn/api/disease"
a=input("询问什么信息：")
params={"word":a}
payload = {}
headers = {
    "User-Agent": 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/141.0.0.0 Safari/537.36 Edg/141.0.0.0',
    "Context-type":"application/json"
}
ta=time.time()
response = requests.get(url, headers = headers, params=params)
tb=time.time()
print(response.text)
print(f"time consumption : %.8f"%(tb-ta))
```
- 请求字段说明

| 参数名 | 传递参数 | 传入位置 | 类型 | 参数说明 |
|--------|----------|----------|------|----------|
| word | 疾病名称 | query | 必传 | 在此传入你要搜索的疾病名称 |

- 访问字段说明

| 字段名 | 信息描述           |
|--------|----------------|
| data | 数据总段数组         |
| acompany | 并发症数组          |
| category | 疾病分类数组         |
| cause | 病因             |
| check | 推荐检查项目         |
| cost_money | 预估费用           |
| cure_department | 治疗科室数组         |
| cure_lasttime | 治疗持续时长         |
| cure_way | 治疗方式数组         |
| cured_prob | 治愈率            |
| desc | 疾病描述           |
| drug_detail | 所有有关药物         |
| get_prob | 患病概率           |
| get_way | 传播途径           |
| name | 病名             |
| prevent | 预防措施描述 |
| recommend_drug | 推荐药物数组 |
| yibao_status | 医保状态表明 |

- 字段减省建议
  - **detail**：以上所有字段导入
  - **simple**：上述中`acompany, category, cause, check, cure_department, cure_lasttime, cure_prob, desc, get_way, name, recommend_drug, prevent, yibao_status`
  - **less**：上述中`category, cause, check, cure_department, desc, get_way, name, recommend_drug, prevent, yibao_status`
  - **desc_only**：上述中`category, cause, check, desc, get_way, name, prevent, recommend_drug`
  - **all_drugs**：上述中`check, drug_detail`
