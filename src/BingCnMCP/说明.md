# Bing中文搜索MCP

ModelScope网址：https://modelscope.cn/mcp/servers/@yan5236/bing-cn-mcp-server

---

- npx用于Stdio格式的部署JSON：
```json
{
  "mcpServers": {
    "bingcn": {
      "args": [
        "bing-cn-mcp"
      ],
      "command": "npx"
    }
  }
}
```
- 安装
```bash
npm install -g bing-cn-mcp
```
或
```bash
npx bing-cn-mcp
```

- 使用（启动）
启动服务器
```bash
bing-cn-mcp
```
或通过npx
```bash
npx bing-cn-mcp
```

- 支持MCP环境中使用
- cursor
```json
{
  "mcpServers": {
    "bingcn": {
      "command": "npx",
      "args": [
        "bing-cn-mcp"
      ]
    }
  }
}
```
- windows
```json
{
  "mcpServers": {
    "bingcnmcp": {
        "command": "cmd",
        "args": [
          "/c",
          "npx",
          "bing-cn-mcp"
      ]
    }
  }
}
```
