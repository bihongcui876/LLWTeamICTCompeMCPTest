## 目前审定可用之API

- BHC876
1. **工具9**：通过病症和关键字查询有关属性和数据对
- - 关键信息：询问关键字、关系名称。
  - 可获得有效信息：**文章id**，**文章信息**，**部分描述**
  - 最大问题：信息并非简练而是完全没有什么东西，且必须知道合适的关系名称。
  - 适合人群：业内医疗人士
  - 适合项目：询问信息查询或者原子项目构建
  - 本项目适合运用：“有什么最新的研究成果或关系信息”

2. **工具2**：一句话分析关联三元组原子关系
- - 关键信息：输入的句子
  - 最大问题：信息为知识图谱的原子信息
  - 本项目适合运用：有关信息对或分子因素反馈

3. **工具3**：关于医学的一句话进行有关分词
- - 关键信息：有关的一句话
  - 本项目适合运用：分词辅助理解，作为内嵌功能

4. **工具1**：所有的原子信息，以至于形成一个关系树
- - 关键信息：实体的名称
  - 本项目适合运用：当今学术界的所有研究内容
